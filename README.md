# DRSTool
DXVK-NVAPI DRS Settings Wrapper - A wrapper for easily creating DXVK_NVAPI_DRS_SETTINGS= commands, it made in the sense of nvidia profile inspector tool for Windows, but for Linux. Tried to make it easily used and intuitive. Create Nvidia driver profiles for your games and apps. Configurations can be saved under profiles.

Dependencies: >=python-3.7 and >=pyside-6.10
